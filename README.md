# India-Crime-Analysis
Power BI dashboard for analyzing Indian crime data using DAX, Power Query, KPI cards, and interactive visualizations.

## Project Overview
An interactive Power BI dashboard analyzing crime statistics across Indian states. The dashboard provides insights into property crimes, murder victims, and rape victims with dynamic filtering and KPIs.

## Tools Used
- Power BI
- Microsoft Excel
- Power Query
- DAX

## Dashboard Features
- Interactive Year Filter
- Crime Type Filter
- KPI Cards
- Treemap
- State-wise Analysis
- Victim Analysis
- Property Crime Analysis

## KPIs
- Total Property Stolen
- Property Recovered
- Stolen Property Value
- Recovered Property Value
- Total Murder Victims
- Victims by Age Group
- Rape Cases

## Files Included
- Power BI Dashboard (.pbix)
- Dataset (.xlsx/.csv)
- Dashboard Screenshots
- Demo Video
